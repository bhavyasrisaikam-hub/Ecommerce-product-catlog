# E-Commerce Product Catalog

## Overview

A responsive E-Commerce Product Catalog built using HTML, CSS, and JavaScript. The application allows users to browse products, search items, filter categories, view detailed product information, add products to a shopping cart, and manage cart items using Local Storage.

---

## Features

* Product Listing using Fake Store API
* Product Search Functionality
* Category-Based Filtering
* Product Details Page
* Shopping Cart System
* Add and Remove Cart Items
* Local Storage Persistence
* Responsive Design
* Interactive Hover Effects
* Modern User Interface

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fake Store API
* Local Storage

## Installation

1. Clone the repository

git clone 

2. Open the project folder in Visual Studio Code.

3. Install the Live Server extension.

4. Right-click index.html and select "Open with Live Server".

---

## Usage

* Browse products on the homepage.
* Search products using the search bar.
* Filter products by category.
* Click a product to view details.
* Add products to the cart.
* Remove products from the cart.
* Cart data remains available after page refresh using Local Storage.
